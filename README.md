# PowerBI-Investment-Analysis-Dashboard
📊 Investment Analysis Dashboard using Power BI
📌 Overview

This project presents an end-to-end Power BI Investment Analysis Dashboard developed during my Power BI Internship at Cognifyz IT Solutions. The dashboard analyzes investor behavior using interactive visualizations, KPIs, DAX measures, and Power Query transformations.

📌 Key Insights
Mutual Funds emerged as the most preferred investment avenue.
Capital Appreciation was the leading savings objective.
Better Returns was the primary reason for investing in Mutual Funds.
Financial Consultants were the most trusted source of investment information.
Most respondents preferred a 3–5 year investment duration.
